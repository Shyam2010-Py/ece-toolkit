📜 Changelog

All notable changes to ECE Toolkit will be documented in this file.

The project follows a simple versioning system:

- Major Version → Big features or redesigns
- Minor Version → New tools and improvements
- Patch Version → Bug fixes and small changes

---

🚀 Version 1.0.0

Initial Public Release

📅 Release Date: 9 June 2026

✨ Features Added

⚡ Ohm's Law Calculator

- Calculate Voltage (V)
- Calculate Current (I)
- Calculate Resistance (R)
- Input validation
- Unit display support

🎨 Resistor Color Code Calculator

- 4-Band resistor support
- Resistance value calculation
- Tolerance calculation
- Live color preview

🔄 Electronics Unit Converter

Supported conversions:

- mV ↔ V
- μA ↔ mA ↔ A
- Ω ↔ kΩ ↔ MΩ
- Hz ↔ kHz ↔ MHz

📊 Attendance Calculator

- Attendance percentage calculation
- Classes that can be missed before reaching 75%
- Classes required to reach 75%
